# 🧾 Task Management API (fastapi - postgres - docker - systemd)

---

## 🚀 how to install :

### Raw Clone Mode
``` bash
git clone https://github.com/alisharify7/Task-Sample-FioTrix
cd Task-Sample-FioTrix
sudo chmod +x ./scripts/install.sh
sudo ./scripts/install.sh
```
### Or via curl
``` bash
curl -fsSL https://raw.githubusercontent.com/alisharify7/Task-Sample-FioTrix/refs/heads/main/scripts/install.sh -o install.sh && sudo chmod +x install.sh && sudo bash install.sh
```

## 🐳 Running the Project with Docker

    docker compose up --build
